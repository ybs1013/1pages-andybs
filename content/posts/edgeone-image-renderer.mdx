## 🚀 快速上手教程

### 1. 创建模板

```javascript
// 创建基础模板
const template = new ImageRenderer({
  width: 1200,
  height: 630,
  background: '#ffffff'
});

// 添加文本元素
template.addText({
  content: '限时特惠',
  font: 'Arial',
  size: 48,
  color: '#ff0000',
  position: { x: 100, y: 100 }
});
```

### 2. 动态替换内容

```javascript
// 定义变量
const variables = {
  title: '${title}',
  price: '${price}',
  date: '${date}'
};

// 批量生成
async function generateImages(data) {
  const results = await Promise.all(
    data.map(item => 
      template.render({
        title: item.title,
        price: item.price,
        date: item.date
      })
    )
  );
  return results;
}
```

### 3. 分发使用

```javascript
// 获取图片URL
const imageUrl = await template.getShareUrl({
  expires: '2h',
  watermark: true
});

// 使用CDN加速
const cdnUrl = `https://cdn.edgeone.ai/${imageUrl}`;
```

## 🎁 最佳实践

### 电商场景示例

```javascript
// 创建商品促销图
const createPromoImage = async (product) => {
  const template = new ImageRenderer({
    template: 'promo-template',
    variables: {
      productName: product.name,
      price: product.price,
      discount: product.discount
    }
  });
  
  return await template.render();
};
```

### 社交分享示例

```javascript

    template: 'social-card',
    variables: {

      content: userData.message

  });
  
  return await template.render();

```
